TWITTER FIND AUTHORS

"Because sometimes you'd like to know who is the author of the (re)tweet"

Comments:

This application extract the URLs of your Tweets, and uses the search engine to find the 1st tweet that posted that link.. easy ^^

Requeriments:

Create a twitter application for getting the required keys to interact with the Twitter API, and write the keys in the python script.

Usage example:

$ ./twitter_find_authors.py
